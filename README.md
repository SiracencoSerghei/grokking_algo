# Grokking Algorithms – Practice Repo

This repository contains my implementations, notes, and exercises based on the book **"Grokking Algorithms" by Aditya Bhargava**.

The goal is to build strong fundamentals in algorithms and problem solving while practicing clean code in Go.

---

## 📚 What’s inside

- Binary Search
- Arrays & Basic Data Structures
- Recursion
- Sorting Algorithms
- Hash Tables
- Breadth-First Search (BFS)
- Dijkstra’s Algorithm
- Greedy Algorithms
- Dynamic Programming (basics)

Each topic includes:
- Simple explanations
- Go implementations
- Practice exercises

## 🧠 Purpose of this repo

- Strengthen algorithmic thinking
- Learn Big-O complexity intuitively
- Build solid foundation for technical interviews
- Practice writing clean Go code
